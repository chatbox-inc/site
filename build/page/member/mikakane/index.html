代表取締役後藤 知宏

phpfrontendfull-stack

株式会社chatbox代表。関西フロントエンドUG代表。

「炎上プロジェクトの火消し役」として、多くの制作企画の立て直しに携わり、プロジェクトの管理手法を学ぶ。

技術面でも多様なコミュニティ活動に参加し、2015年01月には関西最大級のフロントエンドエンジニアコミュニティを発足。

PHPやフロントエンド周りのWeb系技術を中心に制作業務を行う傍ら、セミナー・登壇活動とマルチに活動の幅を広げている。

chatboxでは一緒に働くスタッフ、フリーランスメンバーを募集しています。

学生、未経験者でも歓迎。Web制作にまつわる基礎技術から最新の技術まで、スタッフが丁寧にサポートします。

chatbox メンバー特典

  • 自社開発から始める実務経験サポート
  • プロ向け有償開発ツールの提供
  • 書籍購入補助
  • 就活などに便利なポートフォリオページの掲載

こんな方を募集しています

  • Web制作を学びたい人
  • 独学では学べない実務経験を積みたい
  • コミュニティ活動に興味がある
